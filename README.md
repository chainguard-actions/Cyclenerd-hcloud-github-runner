# Cyclenerd/hcloud-github-runner

A GitHub Action to automatically create Hetzner Cloud servers and register them as self-hosted GitHub Actions runners.

Hardened by [Chainguard](https://www.chainguard.dev) from the upstream action at [https://github.com/Cyclenerd/hcloud-github-runner](https://github.com/Cyclenerd/hcloud-github-runner).
